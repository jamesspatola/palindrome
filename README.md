Palindrome Code Evolution
=========================
This code tests a typed line to see if it is a palindrome. Also it was done 3 ways that version control should show.

## Getting Started
Compile the Pal.java file to a bytecode. Then run the java.class file.

## Prerequisites
You'll need the java development kit installed on your system to compile then run the code.

## How to compile and run java code
Type "javac Pal.java" in the directory with the file. Then type "java Pal" to run pal.class.

## How to test for palindrome
After running the program type your word/phrase/sentence and hit enter.
